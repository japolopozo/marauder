# ESP32 Marauder - Herramienta Educativa de Ciberseguridad

## 📌 Descripción
Este proyecto es una implementación educativa del ESP32 Marauder para analizar redes WiFi y Bluetooth en entornos controlados.

Está diseñado para ayudar a los estudiantes a comprender la seguridad inalámbrica, el hacking ético y las vulnerabilidades de red utilizando hardware de bajo costo.

## ⚙️ Características
- Escaneo de redes WiFi (SSID, BSSID, RSSI, Canal)
- Detección de dispositivos Bluetooth
- Visualización en pantalla OLED (SSD1306)
- Control mediante comandos CLI
- Registro de datos con SPIFFS
- Multitarea con FreeRTOS

## 🧰 Requisitos de hardware
- Módulo de desarrollo ESP32
- Pantalla OLED SSD1306 (128x64)
- Cable USB
- Protoboard (opcional)

## 💻 Requisitos de software
- Arduino IDE / PlatformIO
- Librerías ESP32
- Librerías OLED (Adafruit SSD1306)

## 🚀 Cómo usar
1. Carga el código en el ESP32
2. Abre el Monitor Serie
3. Usa los comandos:

- SCAN_WIFI

- SCAN_BT
- PAUSE

- RESUME

## ⚠️ Aviso legal
Este proyecto tiene fines educativos únicamente. Úselo únicamente en entornos autorizados y controlados.
